# GDB-to-Pars-CLIGEN
Creates CLIGEN par files from ESRI geodatabases containing parameter maps.

## Setup
```bash
-CWD
  |--list.txt
  |--search_stations.txt
  |--pars
  |--wind-strings
    |--*.txt (files containing formatted wind parameter set text blocks)
  |--CCSM4.gdb
  |--CanESM2.gdb
  |--MIROC5.gdb
  |--classes
    |--formatting.py
```
